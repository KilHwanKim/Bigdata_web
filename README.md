## How to Run

### Algorithm

```sh
cd Algorithm
pip install -r requirements.txt
python parse.py
python analyse.py
python visualize.py
```

### PJT

**Backend**

```sh
cd PJT/backend
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py initialize
python manage.py runserver
```

**Frontend**

```sh
cd PJT/frontend
npm install
npm run serve
```
